# todo-list-react
一个响应式布局，基于React实现的待办事项列表网站

## 功能：
+ 添加待办事项
+ 设置已完成待办事项
+ 清除已完成待办事项
+ 根据完成与否显示事项

## 改进方向：
+ 利用localStorage存储数据
